# Incident Response Using the NIST Cybersecurity Framework

This project demonstrates how the NIST Cybersecurity Framework (CSF)
can be applied to respond to a simulated security incident.

## Framework Used
- Identify
- Protect
- Detect
- Respond
- Recover

## Skills Demonstrated
- Incident response analysis
- Risk assessment
- Security documentation
- Framework-based decision making

## Notes
This case study is based on a simulated scenario completed as part of
professional cybersecurity training and rewritten in my own words.
